Introduction to SPNPS

Smart Prediction System for Natural Products Structure

The SPNPS consists of three modules:

  • Build an AI structure prediction module by the deep neural network. The structures and corresponding building blocks of a class of NPs were used as the training set for AI program to learn the underlying biosynthetic rules;
  • According to the literature or preliminary experiments, determine the building blocks contained in a related target plant, and use the above AI program to predict the NPs that these building blocks may generate (called virtual NPs);
  • Collect the experimental LC-MS/MS data of the extract of the target plant, which was compared with the calculated MS/MS data of the virtual NPs. Then screen out the NPs that may actually exist in plants (called predicted NPs).
